# 机器人触摸交互与表情映射规范 (Touch Interaction Spec)

## 一、 交互概述

为了增强机器人的宠物化性格和情感连接，本规范定义了基于电容触摸/物理敲击的 3 种核心交互维度，并将底层的眼球动效与具体的触摸行为进行强绑定。

## 二、 核心触摸交互维度

### 1. 短摸 / 轻拍 (Single Tap / Short Touch)
**交互特征**：用户轻轻拍一下或者摸一下就缩手，通常是为了“唤醒”、“打招呼”或“试探”。

**专属表情映射**：
*   **纳尼？！ (What)** `[ID: 10]`：眼珠平淡收回后突然意识到被摸了，瞬间砸回原方向并夸张放大。作为“突然被拍了一下”的机警反应。
*   **摇摇 (Toon Bounce)** `[ID: 8]`：果冻弹性的物理错觉，极其适合物理敲击的交互反馈，显得肉感Q萌。
*   **震惊 (Shock)** `[ID: 13]`：死死盯住正前方伴随极高频微颤。适合表现有点“受惊”的轻拍反馈。
*   **蛮王冲撞 (Zip)** `[ID: 211]`：向左上方蓄力并伴随剧烈抖动，随后瞬间瞬移到右下方目标位置。短摸触发的强反馈。

### 2. 长摸 / 持续抚摸 (Long Press / Stroke)
**交互特征**：用户把手放在设备上持续超过 1~2 秒，或者来回抚摸。这是建立情感连接、表达喜爱的动作。

**专属表情映射**：
*   **坠入爱河 (Love Struck)** `[ID: 25]`：瞳孔缓慢放大，像没有重力一样波浪线向上漂浮。契合被舒服地抚摸时“飘飘欲仙”的沉醉感。
*   **星星眼 (Star Eyes)** `[ID: 32]`：巨大的水汪汪眼眸，内部闪烁星光。适合在持续抚摸几秒后，给用户的终极正向情绪反馈（卖萌/崇拜）。
*   **委屈 (Puppy Dog)** `[ID: 20]`：眼珠靠拢在顶部中间微微晃动，像小狗求抚摸一样，适合手刚放上去时的撒娇状态。

### 3. 狂戳 / 连续敲击 (Double Tap / Rapid Poke)
**交互特征**：用户在短时间内连续点击，交互带有一定的“破坏性”或“挑逗性”。

**专属表情映射**：
*   **眩晕 (Bonked)** `[ID: 9]`：两只眼珠在眼眶里画圈，相位错开。被戳晕了的经典物理反馈。
*   **弹弹 (Pinball)** `[ID: 12]`：瞳孔像弹珠一样在眼眶内部直线高速反弹。体现被连续敲击后“脑壳嗡嗡作响”的混乱感。
*   **愤怒 (Anger)** `[ID: 19]`：死死盯住前方，快速向中间靠近（斗鸡眼），伴随剧烈震颤蓄力，随后双眼失去同步，向眼眶外侧进行一次极高频的狂暴撞击并被弹回，最后缓慢平息。极具破坏力和躁动感，作为狂戳后的无能狂怒反馈。
*   **大的来了 (Big eyes)** `[ID: 30]`：斗鸡眼向下压，震颤到极限后爆发。狂戳的终极警告动作。
